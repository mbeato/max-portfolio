    The About section has 4 images using `fill` layout without `sizes` props. Without `sizes`, Next.js cannot generate proper `srcset` and downloads oversized images.

    1. **Me.jpg (portrait, row-span-2, left column)**:
       Add `sizes="(max-width: 640px) 45vw, (max-width: 1024px) 18vw, 200px"` to the Image component at ~line 120.

    2. **Climbing.jpg (top right)**:
       Add `sizes="(max-width: 640px) 45vw, (max-width: 1024px) 18vw, 200px"` to the Image component at ~line 132.

    3. **Lifting.jpg (bottom right top)**:
       Add `sizes="(max-width: 640px) 45vw, (max-width: 1024px) 18vw, 100px"` to the Image component at ~line 143.

    4. **Cats.jpg (bottom right bottom)**:
       Add `sizes="(max-width: 640px) 45vw, (max-width: 1024px) 18vw, 100px"` to the Image component at ~line 153.

    **Per PERF-01 (D-01, D-04): fix About photo grid for mobile at 375px.**

    The photo grid currently uses fixed `minHeight` values designed for the 40% desktop column. At 375px with px-6 padding, the grid is only 363px wide, making each column ~179px. The `minHeight: '400px'` on the portrait creates an excessively tall section on mobile.

    Fix the `minHeight` values to scale responsively:

    5. **Portrait container (row-span-2)** at ~line 118:
       Change `style={{ minHeight: '400px' }}` to `style={{ minHeight: 'clamp(250px, 55vw, 400px)' }}`

    6. **Climbing container** at ~line 130:
       Change `style={{ minHeight: '195px' }}` to `style={{ minHeight: 'clamp(120px, 27vw, 195px)' }}`

    7. **Lifting container** at ~line 142:
       Change `style={{ minHeight: '93px' }}` to `style={{ minHeight: 'clamp(56px, 12vw, 93px)' }}`

    8. **Cats container** at ~line 152:
       Change `style={{ minHeight: '93px' }}` to `style={{ minHeight: 'clamp(56px, 12vw, 93px)' }}`

    The `clamp()` values ensure:
    - At 375px: portrait is ~206px (reasonable), climbing is ~101px, lifting/cats are ~45px each
    - At 768px: portrait is ~400px (capped), climbing is ~195px (capped), lifting/cats are ~92px
    - At 1024px+: all values hit their max, matching current desktop appearance exactly
